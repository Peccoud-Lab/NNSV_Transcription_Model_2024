This repository contains the code used for the article "The Transcriptional Gradient in Negative-Strand RNA Viruses Suggest a Common RNA Transcription Mechanism"

DOI: 10.1101/2024.11.11.623041

Notebook1 contains the python code for fitting the RAM model and performing MCMC. The outputs of this notebook are the parameter fits and MCMC statistics in Folder1 and Folder2. The data in these folders was used to make Table S1 and the images used to generate all panels in figure 4.

Notebook2 contains the python code for making predictions of transcriptional gradients for gene-shuffled variants for both the Stop Start and RAM Models.

Markdown1 contains the R Markdown code for fitting the linear regression model connecting polymerase processivity to genome length. It also generates the image used in figure 5.

Markdown 2 contains the code for evaluating the predictions of gene-shuffled variants for the Stop Start and RAM models. This code generates the panels in figure 6.
